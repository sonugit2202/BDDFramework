This repository contains an end-to-end test automation framework built using **Playwright + Cucumber (BDD)**.

---

 # Tech Stack

- Playwright
- Cucumber (BDD)
- JavaScript (Node.js)
- Page Object Model (POM)

---

# Project Structure

BDDFramework/
│── features/ # Feature files (Gherkin)
│── step_definitions/ # Step definitions
│── pages/ # Page Object classes
│── utils/ # Utilities (test data, helpers)
│── tests/ # Test runners
│── playwright.config.js # Playwright config
│── package.json

## 🚀 Features

- BDD-style test scenarios using Gherkin
- Page Object Model design
- Reusable step definitions
- API + UI automation support (if applicable)
- Scalable and maintainable structure
